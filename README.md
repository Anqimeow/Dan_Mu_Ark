# 旦暮方舟 🚢 - 高校期末周 AI 辅助 Agent

## 项目概述

「旦暮方舟」是一款聚焦高校学生期末周高压场景的 **“情感 + 学业” 双驱动 AI 辅助工具**，以 “减负・诊断・治愈” 为核心设计哲学，精准解决期末复习阶段的认知过载、决策瘫痪、情感隔绝三大核心痛点，为学生提供从知识梳理、盲区定位到心理修复的全栈式护航。

## 核心痛点解决



| 核心困境 | 具体问题                                    | 解决方案                             |
| ---- | --------------------------------------- | -------------------------------- |
| 认知过载 | 海量复习资料（PPT、教材、论文）难以快速提取核心考点，复习效率低下      | AI 智能提炼核心考点，将长篇资料转化为精准复习清单       |
| 决策瘫痪 | 多门高学分、高难度科目并行，缺乏科学优先级排序，陷入 “不知先学哪门” 的焦虑 | 科学科目优先级推荐算法，基于学分权重、难度系数、掌握程度智能排序 |
| 情感隔绝 | 闭关复习导致社交孤岛，心理压力缺乏即时疏导，易产生习得性无助          | 实时情感能量监测 + 校园共鸣回血机制，提供即时心理缓冲     |

## 核心价值

### 对高校学生



1. **效率倍增**：AI 深度处理复习资料，3 万字教材快速转化为核心复习清单，搭配智能自测系统精准定位知识盲区，复习效率提升 40%+；

2. **心理护航**：填补 “深夜崩溃无反馈” 的空白，通过情感监测与校园化共鸣机制，有效缓解期末焦虑，避免习得性无助；

3. **场景适配**：深度贴合高校专属语境（如复旦特定学科重点、校园打卡点情感共鸣），解决通用 AI 工具的 “场景脱节” 与 “幻觉” 问题，适配性更强。

### 对业务 / 社会



1. **模式创新**：探索 AI 在教育场景中 “生产力工具 + 情感陪伴” 的融合落地模式，为 EdTech 领域提供创新范本；

2. **跨校复用**：核心架构支持快速适配不同高校，可低成本衍生 “交大版”“同济版” 等定制化版本，覆盖更多高校学生群体；

3. **轻量化部署**：适配云端低成本环境（2vCPU/8G 内存），无需本地安装，通过网页 / 小程序即可使用，降低用户使用门槛。

## 适用场景



* 高校学生期末周多科目并行复习

* 海量复习资料梳理与核心考点提取

* 复习计划制定（科目优先级排序）

* 期末高压下的心理压力疏导

* 校园场景化学习陪伴与情感支持

## 技术特性



* 多模态资料处理（支持 PPT、PDF、TXT 等格式教材 / 资料解析）

* 基于高校学科特点的 AI 精准提炼算法

* 情感计算模型（适配学生期末心理状态识别）

* 轻量化云端部署架构（低成本、高可用性）
